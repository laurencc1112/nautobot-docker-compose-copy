# nautobot-docker-compose
Docker-compose for Nautobot, eventually to be moved into the nautobot repo


Build should contain:

1. Nautobot Server
2. Nautobot Worker
3. Redis Caching
4. Redis Worker Queue
5. Postgres
6. NGINX Webserver (SSL Termination)
